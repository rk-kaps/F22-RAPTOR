# Foam Board RC Plane

A lightweight radio-controlled aircraft built using foam board, a BLDC motor, two servos, and a standard RC control system.

## Overview

This project aims to design and construct a functional RC aircraft using low-cost and easily available components. The aircraft is built primarily from foam board and uses a brushless DC (BLDC) motor for propulsion. Control is achieved through two servo-actuated control surfaces: the elevator and rudder.

The project explores the fundamentals of aircraft design, aerodynamics, propulsion systems, radio communication, and flight stability.

## Features

- Foam board airframe
- BLDC motor propulsion system
- Electronic Speed Controller (ESC)
- Elevator and rudder control
- Wireless RC transmitter and receiver
- Lightweight and low-cost design
- Modular construction for easy repairs and modifications

## Build Process

1. Design the airframe.
2. Cut all foam board components.
3. Assemble fuselage, wings, and tail surfaces.
4. Install motor mount and propulsion system.
5. Mount servos and control linkages.
6. Install receiver and battery.
7. Configure transmitter and receiver.
8. Verify center of gravity.
9. Perform ground tests.
10. Conduct maiden flight.
